# Agri AI

A Django-based agricultural assistance system that predicts soil quality and provides crop-related insights using machine learning models.

## Tech Stack
- Python
- Django
- HTML/CSS
- Machine Learning (scikit-learn)

## Setup Instructions
1. Clone the repository
2. Create and activate a virtual environment
3. Install dependencies:
   pip install -r requirements.txt
4. Run migrations:
   python manage.py migrate
5. Start server:
   python manage.py runserver

## Notes
- media/uploads is ignored as it contains runtime-generated files
- db.sqlite3 is not tracked
